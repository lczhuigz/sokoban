# 🎮 推箱子 Sokoban

经典推箱子游戏，基于 Vue 3 + Vite 构建。30 个精心设计的关卡，支持自定义关卡编辑器。

## 功能

- 🧩 **30 个关卡** — 从入门到大师，逐级解锁
- ↩️ **无限撤销** — 随时回退，不怕走错
- ⏱ **计时挑战** — 记录用时，突破最佳成绩
- ✏️ **关卡编辑器** — 自由创建自定义关卡
- 📱 **移动端适配** — 触屏滑动 + 虚拟方向键
- 💾 **自动保存** — 进度和最佳记录本地存储

## 操作

| 操作 | 按键 |
|------|------|
| 移动 | `↑↓←→` 或 `WASD` |
| 撤销 | `Ctrl + Z` |
| 重置 | `R` |
| 关卡选择 | `Esc` |

移动端支持滑动手势和虚拟方向键。

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建
npm run build

# 预览构建结果
npm run preview
```

## 技术栈

- [Vue 3](https://vuejs.org/) — Composition API
- [Vite](https://vitejs.dev/) — 构建工具
